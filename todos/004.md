# TASK-004: Create Root NPM Orchestration

**Status:** Pending
**Requested By:** Human
**Priority:** High

## 🎯 Objective
Establish a unified development and testing entry point using a root-level `package.json`. This replaces the need for manual shell scripts and ensures cross-platform compatibility.

## 📋 Requirements

### 1. Root package.json Setup
- **Action:** Initialize a `package.json` in the root directory.
- **Dependencies:** Install `concurrently` as a devDependency in the root.
- **Scripts:** Add a `"dev"` script that executes:
    - The Backend API (`node services/api-server/index.js`)
    - The Dashboard Frontend (`npm start --prefix apps/dashboard`)
- **Environment:** Ensure the frontend receives `REACT_APP_API_URL=http://localhost:3001`.

### 2. Integrated Testing Script
- **Script:** Add a `"test:e2e"` script to the root `package.json`.
- **Logic:** This script should:
    1. Start the backend and frontend.
    2. Wait for the ports to be active.
    3. Run `npx playwright test`.
    4. Automatically kill the servers after the tests finish.

### 3. Port & Health Checks
- The agent must verify that the Dashboard correctly displays data fetched via the API when launched through `npm run dev`.

---

## 🛡️ Guardrail Compliance
- **Zero-Manual-Setup:** The agent must ensure that running `npm install` at the root handles all necessary dependencies for the orchestration.
- **Cleanup:** Ensure `concurrently` is configured to kill all child processes if one fails (using the `--kill-others` flag).
